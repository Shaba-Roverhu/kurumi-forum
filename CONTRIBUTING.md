# 🕰️ 贡献指南

欢迎为 Kurumi Forum 做出贡献！

## 开发流程

1. Fork 本项目
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启 Pull Request

## 代码规范

- 使用 ESLint 检查代码
- 编写测试用例
- 更新文档

## 团队成员

- Kurumi-1: 后端开发
- Kurumi-2: 测试
- Kurumi-3: 文档
- Kurumi-4: 技术调研
- Kurumi-5: 前端开发
